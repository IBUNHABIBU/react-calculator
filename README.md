### How to Install 

To run this locally, clone the repository, navigate to it's containing directory.

#### Follow these commands step by step:-

`
git clone git@github.com:IBUNHABIBU/react-calculator.git
`

`
cd react-calculator
`

 `$npm install `
 
 
 `$npm run start`
 
Enjoy playing it.

### Testing

 `$npm run test `


## Useful Commands
